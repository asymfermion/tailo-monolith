# Tailo Mobile

The Tailo iOS-first React Native app (Expo). It handles photo scanning, local event building, and the timeline — built to work offline and before sign-up.

See the **[developer guide](../../docs/DEVELOPER.md)** for setup and run instructions.

## Brand assets

Source artwork lives in `assets/brand/`:

| File                         | Use                                        |
| ---------------------------- | ------------------------------------------ |
| `app-icon.png`               | App icon, splash, Android adaptive icon    |
| `favicon.png`                | Web favicon (48×48, derived from app icon) |
| `logo-lockup-stacked.png`    | Marketing / store (icon + wordmark)        |
| `logo-lockup-horizontal.png` | Marketing / wide layouts                   |

Expo paths are configured in `app.json`.
