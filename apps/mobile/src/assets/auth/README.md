# Tailo Auth Hero Assets

Use these files to implement the printed-photo auth hero as layered UI.

## Recommended layer assets

- `tailo-wordmark-dark-transparent.png` — transparent Tailo wordmark. Render directly on the warm paper background.
- `apple-logo-black.png` — black Apple mark from Apple AuthKitUI resources for auth buttons.
- `auth-photo-onboarding-main-beach.png` — main printed-photo image for onboarding.
- `auth-photo-login-main-clean.jpg` — main printed-photo image for login.
- `auth-photo-mini-beach.jpg` — small overlapping polaroid image.
- `google-g-logo.png` — official Google Identity multi-color G mark for auth buttons.
- `torn-paper-edge-ivory.png` — optional torn-paper overlay at bottom of the hero photo.

## Reference-only files

Reference composites from the source handoff were removed after implementation so they are not bundled as unused app assets. The app should continue to compose the hero from the layered assets above.
