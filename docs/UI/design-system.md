# Tailo Design System

## Brand Personality

Warm, calm, emotional, premium, photo-first.

## Layout

- 20-24pt horizontal padding
- Generous whitespace
- Rounded corners (16-24) for photo cards and large panels
- Pill controls for primary actions, segmented choices, and small chips
- Minimal borders
- Photo-first screens with warm paper-like negative space

## Components

### Memory Card

- Clean hero image without a full transparent text cover
- Supporting image strip
- Title below the hero image
- Story text
- Date and photo count

### Floating Navigation

- Story
- Pet
- Settings
- Glass morphism effect
- Pet avatar in center

### Buttons

Primary: filled ink
Secondary: warm outline or text

### Segmented Controls

- Selected segment uses filled ink with light text
- Unselected segments sit on warm ivory or soft taupe

## Color Palette

The three sketch references use a warm, almost photo-book palette with black/ink controls. Do not use green or coastal blue as the product accent unless the sketches are revised.

Background #FBF7F1 — warm ivory
Surface #FFFDF9 — warm white
Text #151412 — ink
Accent #151412 — ink for primary buttons, selected tabs, selected segments, and active controls
Muted #6F665D — warm gray-brown secondary text
Border #E7DDD2 — soft taupe
Divider #D8CBBE — timeline and section rules
