# Tailo Design System

## Brand Personality

Warm, calm, emotional, premium, photo-first.

## Layout

- 20-24pt horizontal padding
- Generous whitespace
- Rounded corners (16-24)
- Minimal borders

## Components

### Memory Card

- Clean hero image without a full transparent text cover
- Supporting image strip
- Title below the hero image
- Story text
- Date and photo count

### Floating Navigation

- Story
- Pet
- Settings
- Glass morphism effect
- Pet avatar in center

### Buttons

Primary: filled accent color
Secondary: outlined or text

## Color Palette

Background #F7F5F2
Text #1C1C1A
Accent #5A738A — muted coastal blue
Muted #6B6B66
Border #E8E4DE
