# Tailo Product Principles

## Mission
Help pet owners rediscover, preserve, and relive their pet's story through memories hidden inside their photo library.

## Primary Journey
Install
→ Start with Photos
→ Permission
→ Discovery
→ Pet Confirmation
→ First Memory
→ Story Feed
→ Emotional Attachment
→ Create Account

## Account Philosophy
Anonymous-first.
Accounts exist for backup, sync, and premium features.

## Success Metric
The first time a user sees a meaningful memory generated from their own photos.
