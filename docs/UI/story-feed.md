# Story Feed

## Philosophy

The feed is the product.

Users should understand a memory without opening detail.

## Structure

Life with Link

428 memories · 2 years together

Visual language:

- Warm ivory background
- Ink selected states and primary controls
- Soft taupe borders/dividers
- Rounded, photo-led memory cards

Memory:

- Hero image
- Title
- Supporting images
- Story text
- Date
- Photo count

## Avoid

- Timeline rails
- Dashboard cards
- Dense metadata
- Full-image transparent text covers on memory photos
