# Tailo Style Guide

## Product Identity

Tailo is a private, memory-first pet story app.

### Core Principles

- Show value before commitment
- Photos are the interface
- Story over database
- Fast path to first memory
- Private by default

## Visual Direction

Inspired by the three UI sketches, Apple Photos Memories, Apple Journal, and premium photo books.

The visual language is warm ivory, ink controls, soft taupe borders, rounded photo cards, and quiet brown-gray secondary text. Avoid green or blue accent drift unless the sketches are intentionally revised.

## Navigation

- Story
- Pet
- Settings

Floating glass navigation pill. Center tab represents the active pet.

## Story Feed

- Multi-photo memory cards
- Hero image + supporting images
- Clean hero photos without full-image transparent text covers
- Story text visible in feed
- Photos first, story second, metadata third
- Ink primary controls and selected states
- Warm ivory page background with soft taupe dividers

## Pet Profile

Character page, not settings page.
Hero photo, story stats, highlights, about, life timeline.

Use the sketch language: a large photo-led hero, soft glass stat strip, ink selected segments, warm outline chips, and minimal borders.

## Editing

Auto-save by default. No Save button.
