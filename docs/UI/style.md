# Tailo Style Guide

## Product Identity

Tailo is a private, memory-first pet story app.

### Core Principles

- Show value before commitment
- Photos are the interface
- Story over database
- Fast path to first memory
- Private by default

## Visual Direction

Inspired by Apple Photos Memories, Apple Journal, premium photo books.

## Navigation

- Story
- Pet
- Settings

Floating glass navigation pill. Center tab represents the active pet.

## Story Feed

- Multi-photo memory cards
- Hero image + supporting images
- Clean hero photos without full-image transparent text covers
- Story text visible in feed
- Photos first, story second, metadata third

## Pet Profile

Character page, not settings page.
Hero photo, story stats, highlights, about, life timeline.

## Editing

Auto-save by default. No Save button.
