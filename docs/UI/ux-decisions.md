# Tailo UX Decisions

## Onboarding

1. Welcome screen
2. Native photo permission
3. Discovery
4. Story feed

## Welcome Screen

Primary CTA: Start with My Photos
Secondary: Apple, Google, Sign In

## Story Feed

Users should understand a memory without opening details.

Each memory:

- Hero image
- Title
- Supporting images
- Story text
- Date
- Photo count

Hero photos should stay visually clean. Do not place a full transparent text cover over every memory image; keep story/title text in the card body and metadata below.

## Pet Profile

- Hero photo
- Story stats
- About
- Highlights
- Life timeline
